# Desafio One Piece - CRUD do Sanji

Um projeto simples em HTML, CSS e JavaScript que simula um sistema de controle de estoque de comidas, inspirado no cozinheiro mais brabo dos mares: Sanji.

## Funcionalidades
O sistema possui as seguintes funcionalidades:

- ✅ Cadastrar comida no estoque
- 🔍 Pesquisar uma comida
- 📊 Verificar quantidade de itens cadastrados
- 📦 Exibir todo o estoque
- ✏️ Alterar nome de uma comida
- ❌ Remover comida do estoque
- 🧹 Limpar campos do formulário
